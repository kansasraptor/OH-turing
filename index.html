API (Application Programming Interface)
It refers to a tool, or library, that assists developers in writing code that interfaces with other software. These can range from the low-level tools that allow Windows and OS X applications to use aspects of the underlying operating system, to those powering the apps on your phone. In short, it defines a way in which a computer program communicates with another computer program. makeuseof.com
